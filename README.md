## About The Project

**EaziKing Collections** is a modern, elegant e-commerce platform showcasing premium men's clothing. Built with a focus on user experience, accessibility, and performance, this website represents our commitment to quality fashion and exceptional online shopping.

### Live Demo
[Visit EaziKing Collections →](https://eazikingcollection.github.io/eazikingcollection)

---

## ✨ Features

- **Clean & Professional Design** — Modern UI with premium feel
- **Fully Responsive** — Optimized for mobile, tablet, and desktop
- **Advanced Product Filtering** — By category, price, and more
- **Shopping Cart System** — Add, remove, and manage items seamlessly
- **Payments Integration** — Real payment processing (Supports Nigerian payment methods cards, mobile money, etc.)
- **Fast & Lightweight** — Built for performance and SEO
- **Accessible** — WCAG-friendly with proper ARIA labels and keyboard navigation
- **Product Detail Pages** — Rich descriptions, size selection, and high-quality images

---

## 🎨 Design Philosophy

We believe clothing should be more than just fabric — it should reflect confidence, elegance, and comfort. Our collection targets the modern gentleman who values quality, style, and attention to detail.

**Primary Colors:**
- Emerald Green (`#0f766e`)
- Teal Accents
- Clean Neutrals & Warm Grays

---

## 🛠️ Tech Stack

- **Frontend Framework**: Bootstrap 5.3 (Responsive & Professional UI)
- **Core Languages**: HTML5, CSS3, Vanilla JavaScript
- **Payments Integration**: Snipcart (Supports Nigerian payment methods)
- **Data Management**: Static JSON files
- **Hosting & Deployment**: GitHub Pages
- **Styling Approach**: Bootstrap + Custom CSS for brand colors (Emerald Green & Teal)
- **Accessibility**: WCAG 2.1 compliant basics (screen reader friendly)
